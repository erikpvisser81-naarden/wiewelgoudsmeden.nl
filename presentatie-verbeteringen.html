Mobiele ervaring

60%+ van je bezoekers zit op een telefoon

Huidige site
  • ✓ Image carousel als hero (werkt goed)
  • ✓ "Bekijk de collectie" CTA onder hero
  • × Alleen "Bekijk de collectie", geen afspraak-CTA in hero
  • × Header verdwijnt niet bij scrollen (neemt ruimte in)
  • × Geen "Plan een afspraak" in het eerste scherm
Nieuwe site
  • ✓ Hero met foto + tagline + 2 CTA knoppen
  • ✓ "Plan een afspraak" direct zichtbaar in hero
  • ✓ Slim menu: verdwijnt bij scrollen, komt terug bij omhoog
  • ✓ Vertrouwensbadges (Meestergoudsmid, Gilde) in beeld
  • ✓ Alle knoppen minimaal 44px (Apple richtlijn)

Vindbaarheid in Google

Wat Google nodig heeft

Huidige site
  • ✓ Paginatitel + beschrijving (Yoast)
  • ✓ Sitemap + robots.txt
  • ✓ ~508 pagina's in Google
  • × Geen deelplaatje (OG/Twitter tags)
  • × Geen "Juwelier" schema (adres, openingstijden)
  • × Geen cookie-melding (verplicht bij Hotjar + GA)
  • × Google Analytics verouderd (geen data sinds juli 2023)
Nieuwe site
  • ✓ Paginatitel + beschrijving op elke pagina
  • ✓ Sitemap (99 pagina's) + robots.txt
  • ✓ Deelplaatje bij WhatsApp/social (OG tags)
  • ✓ Schema.org "Jeweler" met adres, telefoon, openingstijden
  • ✓ Product-schema per sieraad (prijs, materiaal)
  • ✓ Cookie-melding met keuze (accepteren/weigeren)
  • ✓ FAQ-schema (6 veelgestelde vragen)

Productpagina's

Elk sieraad krijgt een volwaardige pagina

Huidige site (WooCommerce)
  • ✓ Eigen pagina per sieraad
  • ✓ Prijs en basisspecs zichtbaar
  • × Geen Product-schema voor Google
  • × Dezelfde CTA ("Stel je vraag") voor elk sieraad
  • × Geen vertrouwenssignalen op de pagina
Nieuwe site
  • ✓ Eigen pagina per sieraad
  • ✓ Prijs, specs én vervaardiging zichtbaar
  • ✓ Product-schema (Google kan prijs + naam tonen)
  • ✓ Slimme CTA: dure sieraden (>€5.000) krijgen "Bel ons" knop
  • ✓ Vertrouwenssignalen onder elke CTA

Kern: de huidige site behandelt elk sieraad hetzelfde. De nieuwe site past de pagina aan op het sieraad: andere knop, andere toon, vertrouwenssignalen eronder.
